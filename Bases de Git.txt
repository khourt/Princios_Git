¿Que es git?
Es una herrameinta que sirve para el versionamiento de codigo (Sistema de control de versiones)
Se define como una "maquina de tiempo" para poder navegar entre diferentes versiones de 


- Crear carpeta > mkdir && cd / mkcd
- Crear archivo > touch / nano
 
- Crear repositorio local
- Inicializar el repositorio > git init
	Podemos inicializar el servicio de git en la carpeta 

- Visualizar estado > git status

- Ver las versiones - git log

- Agregar archivos al seguimiento de git

- Crear repositorio en Git hub

- Subir al repositorio. 

 git log
 git branch -M main

 git remote add origin https://github.com/Tezcatlilpocatl/ComidasFavoritas.git
 git push -u origin main
